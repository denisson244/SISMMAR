SISTEMA DE IRRIGAÇÃO SUSTENTÁVEL MICROCONTROLADO COM MONITORAMENTO E ACIONAMENTO REMOTO

O presente trabalho aborda o conceito de automação para produzir um sistema que comande a irrigação de um cultivo de forma sustentável sem a necessidade de intervenção humana recorrente, fazendo uso dos conceitos das tecnologias disponíveis para proporcionar a pessoas inábeis para o cultivo e que não dispõem de tempo para tal tarefa a possibilidade de plantar vegetais em suas residências, podendo, assim, obter os benefícios que o cultivo residencial oferece. Para execução do projeto foi utilizada a Placa Arduino Nano, a qual  desempenha o papel de controlador do processo, realizando o tratamento de sinais provenientes do meio externo e enviando comando de acordos com as medições feitas. Um sensor de umidade implantado na área de plantio faz o monitoramento do solo que de acordo com uma comparação entre a umidade ideal e a umidade a real, determinando o acionamento de uma bomba d’agua que permitirá a irrigação. Assim que o solo atingir a umidade de referência, o sistema de controle implementado no Arduino Nano interrompe a irrigação desligando a bomba. O diferencial deste projeto é o uso de materiais e baixo custo tanto para a construção do sistema eletrônico quanto para os recipientes de cultivo e de reserva de água. O sistema ainda proporciona, por meio de uma display de LCD, o acompanhamento das informações referentes ao solo em tempo real, bem como o estado da bomba d’agua, além de permitir esse mesmo acompanhamento e o acionamento de funções de forma remota.
